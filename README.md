# Project Overview

This repository is organized as follows:


project-root/
├── data/               # Contains raw and processed datasets
├── notebooks/          # Jupyter/Databricks notebooks for analysis and experimentation
├── src/                # Source code for main application logic and utilities
├── tests/              # Unit and integration tests for the codebase
├── docs/               # Documentation files and resources
├── requirements.txt    # Python dependencies
└── README.md           # Project overview and instructions


## Folder Descriptions

- **data/**: Store all datasets used in the project. Separate raw and processed data as needed.
- **notebooks/**: Interactive notebooks for data exploration, prototyping, and reporting.
- **src/**: Core source code, including modules, scripts, and helper functions.
- **tests/**: Automated tests to ensure code quality and correctness.
- **docs/**: Additional documentation, guides, and references.
- **requirements.txt**: List of required Python packages.
- **README.md**: This file. Provides an overview and setup instructions.
